# Property Listings App

A responsive property listing platform with login, filters, and sorting.

## Features
- Login with any credentials
- Protected routes
- Filter by location & price
- Sort by price or date
- Responsive

## Setup
```bash
npm create vite@latest property-listings -- --template react
cd property-listings
npm install react-router-dom
npm install -D tailwindcss
npx tailwindcss init -p
